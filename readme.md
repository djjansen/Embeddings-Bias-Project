# Environment Set Up and Running
The required Python packages are listed in `requirements.txt`. You can set up your environment by running `pip install -r requirements.txt`.

Some of the local models (e.g., Llama 3.1 8b) will require local downloading and loading model weights. Please see system requirements for this.

`config.ini` (not included) is used for credentials and other variables referenced in `llm-embedding-gender-bias.ipynb`.
